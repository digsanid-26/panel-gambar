# Glosarium Istilah — Panel Gambar Bersuara

---

## Istilah Umum Aplikasi

| Istilah | Definisi |
|---------|----------|
| **Panel** | Satu frame/bingkai gambar yang merupakan bagian dari rangkaian cerita. Mirip satu panel dalam komik. |
| **Panel Cerita** | Kumpulan beberapa panel yang membentuk satu cerita utuh dari awal sampai akhir. |
| **Buble Dialog** | Balon/kotak ucapan yang berisi teks dialog karakter, ditempatkan di atas gambar panel. |
| **Narasi** | Teks deskriptif yang menjelaskan konteks cerita di luar dialog karakter. Biasanya ditempatkan di atas/bawah gambar. |
| **Suara Latar** | Audio berupa musik atau efek suara (*sound effect*) yang mengiringi panel cerita untuk membangun suasana. |
| **Authoring Tool** | Fitur/modul dalam aplikasi yang digunakan guru untuk membuat dan mengedit panel cerita. |
| **Panel Viewer** | Fitur/modul untuk menampilkan dan membaca panel cerita oleh siswa maupun guru. |
| **Gambar Pendukung** | Gambar tambahan di luar gambar utama panel, misalnya peta, infografis kecil, atau objek penjelasan. |

## Istilah Pendidikan & Kurikulum

| Istilah | Definisi |
|---------|----------|
| **Membaca Nyaring (Read Aloud)** | Kegiatan membaca teks dengan suara keras/lantang sehingga dapat didengar oleh orang lain, dengan memperhatikan intonasi, lafal, dan ekspresi. |
| **Fluensi Membaca** | Kemampuan membaca teks dengan akurasi, kecepatan, dan prosodi (ekspresi/intonasi) yang tepat. |
| **Literasi** | Kemampuan seseorang dalam membaca, menulis, dan menggunakan informasi untuk berkomunikasi dan berpartisipasi dalam masyarakat. |
| **Capaian Pembelajaran (CP)** | Kompetensi yang diharapkan dicapai peserta didik pada akhir suatu fase dalam Kurikulum Merdeka. |
| **Kurikulum Merdeka** | Kurikulum nasional Indonesia yang memberikan fleksibilitas kepada satuan pendidikan dan guru dalam merancang pembelajaran. |
| **Fase A** | Fase pembelajaran dalam Kurikulum Merdeka untuk kelas 1–2 SD. |
| **Fase B** | Fase pembelajaran dalam Kurikulum Merdeka untuk kelas 3–4 SD. |
| **Profil Pelajar Pancasila** | Profil lulusan yang ingin dicapai melalui pendidikan Indonesia, mencakup 6 dimensi: beriman, berkebinekaan global, gotong royong, mandiri, bernalar kritis, kreatif. |
| **ADDIE** | Model pengembangan instruksional: *Analysis, Design, Development, Implementation, Evaluation*. |
| **R&D (Research & Development)** | Metode penelitian yang bertujuan menghasilkan produk tertentu dan menguji efektivitasnya. |

## Istilah Teknis (Teknologi)

| Istilah | Definisi |
|---------|----------|
| **PWA (Progressive Web App)** | Teknologi web yang memungkinkan aplikasi web berperilaku seperti aplikasi native (bisa di-install, offline, push notification). |
| **SPA (Single Page Application)** | Arsitektur web di mana seluruh aplikasi dimuat dalam satu halaman HTML dan konten di-update secara dinamis tanpa reload. |
| **SSR (Server-Side Rendering)** | Teknik rendering halaman web di sisi server sebelum dikirim ke browser, meningkatkan performa dan SEO. |
| **API (Application Programming Interface)** | Antarmuka pemrograman yang memungkinkan komunikasi antar komponen sistem (misalnya frontend ↔ backend). |
| **REST** | Arsitektur API yang menggunakan metode HTTP (GET, POST, PUT, DELETE) untuk operasi data. |
| **WebSocket** | Protokol komunikasi dua arah secara real-time antara browser dan server. |
| **Canvas** | Elemen HTML5 yang memungkinkan rendering grafis 2D secara programatis, digunakan untuk editor panel. |
| **MediaRecorder API** | API browser yang memungkinkan perekaman audio/video dari perangkat pengguna (mikrofon/kamera). |
| **Web Audio API** | API browser untuk pemrosesan dan sintesis audio tingkat lanjut. |
| **Service Worker** | Script yang berjalan di background browser, memungkinkan fitur offline, caching, dan push notification. |
| **ORM (Object-Relational Mapping)** | Teknik pemrograman yang memetakan objek dalam kode ke tabel dalam database relasional. |
| **CI/CD** | *Continuous Integration / Continuous Deployment* — praktik otomatisasi proses build, test, dan deploy. |
| **IFP (Interactive Flat Panel)** | Perangkat layar sentuh besar (55"–86") yang digunakan sebagai pengganti papan tulis dan proyektor di ruang kelas. |
| **Responsive Design** | Pendekatan desain web yang membuat tampilan menyesuaikan secara otomatis dengan ukuran layar perangkat. |
| **CORS** | *Cross-Origin Resource Sharing* — mekanisme keamanan browser yang mengontrol akses lintas domain. |
| **CDN (Content Delivery Network)** | Jaringan server tersebar global yang mempercepat pengiriman konten ke pengguna berdasarkan lokasi terdekat. |
| **WCAG** | *Web Content Accessibility Guidelines* — standar aksesibilitas web yang ditetapkan oleh W3C. |
| **ARIA** | *Accessible Rich Internet Applications* — atribut HTML untuk meningkatkan aksesibilitas bagi pengguna screen reader. |

## Istilah Penelitian

| Istilah | Definisi |
|---------|----------|
| **Validasi Ahli** | Proses penilaian kelayakan produk oleh pakar/ahli di bidang terkait sebelum diuji coba ke pengguna. |
| **Uji Coba Terbatas (Small Group Trial)** | Uji coba produk pada kelompok kecil pengguna (10–15 orang) untuk mendapatkan masukan awal. |
| **Uji Coba Lapangan (Field Trial)** | Uji coba produk pada kelompok lebih besar dalam kondisi sebenarnya/nyata. |
| **Pre-test** | Tes yang dilakukan sebelum perlakuan/penggunaan produk untuk mengukur kondisi awal. |
| **Post-test** | Tes yang dilakukan setelah perlakuan/penggunaan produk untuk mengukur perubahan. |
| **Skala Likert** | Skala pengukuran yang menggunakan opsi jawaban berjenjang (misalnya 1–5: Sangat Tidak Setuju hingga Sangat Setuju). |
| **Paired Sample T-Test** | Uji statistik parametrik untuk membandingkan rata-rata dua kelompok data yang berpasangan (misalnya pre-test vs post-test). |
| **MRP (Motivation to Read Profile)** | Instrumen terstandar untuk mengukur motivasi/minat membaca yang dikembangkan oleh Gambrell et al. (1996). |
| **Evaluasi Formatif** | Evaluasi yang dilakukan selama proses pengembangan untuk perbaikan berkelanjutan. |
| **Evaluasi Sumatif** | Evaluasi akhir untuk mengukur efektivitas produk secara keseluruhan. |
| **Sinta** | *Science and Technology Index* — sistem pengindeksan dan akreditasi jurnal ilmiah di Indonesia oleh Kemendikbudristek. |
